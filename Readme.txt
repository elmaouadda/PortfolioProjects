## COVID SQL Data Analysis

This repository contains SQL files for analyzing COVID-19 data. The SQL files include tables for tracking COVID deaths and vaccinations, along with queries for data analysis.

### Contents

- `covid_deaths.sql`: SQL script to create the `CovidDeaths` table for tracking COVID deaths.
- `covid_vaccinations.sql`: SQL script to create the `CovidVaccinations` table for tracking COVID vaccinations.
- `data_analysis.sql`: SQL queries for analyzing the COVID data, such as calculating new cases, deaths, and vaccination rates.

### Usage

1. Create a new database and execute the `covid_deaths.sql` and `covid_vaccinations.sql` scripts to create the required tables.
2. Execute the `data_analysis.sql` script to perform data analysis on the COVID data.
